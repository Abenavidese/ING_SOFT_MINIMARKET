Bienvenido al Sistema de Gestión de Minimarket

Administra productos, proveedores, ventas, categorías y caja con una interfaz moderna y fácil de usar. El sistema integra FastAPI y Angular para ofrecer una experiencia rápida y confiable.

Características Clave

  • Operaciones CRUD completas para productos, proveedores y ventas.
  • Endpoints RESTful documentados con Swagger UI.
  • Validaciones con Pydantic y base de datos con SQLAlchemy.
  • Persistencia en SQLite (desarrollo) y PostgreSQL (producción).
  • Pruebas automatizadas con Pytest.
